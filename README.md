# Tombrown Whatsapp Shop

WhatsApp checkout e-commerce built with Next.js (App Router), TypeScript, Tailwind, Zustand.

## Quickstart
```bash
pnpm install
pnpm dev
# or: npm install && npm run dev
```
Visit: http://localhost:3000

## Customize
- Change **shop name** and **WhatsApp number** in `lib/shopConfig.ts`
- Edit products in `lib/products.ts`
- Tailwind theme in `app/globals.css` (CSS variables)
- Images use remote placeholders; replace with your own URLs or use `/public` and Next/Image

## Build
```bash
pnpm build && pnpm start
```

## Deploy
- Vercel recommended. Import the repo, no extra config needed.
